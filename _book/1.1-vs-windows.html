1.1 和Windows的不同

Linux 和 Windows 的不同

Linux 与 Windows 差别很大，

对于经常使用 Windows 的用户，Linux 有以下优点：

  • 免费
  • 不需要安装杀毒/安全软件（相对而言）
  • 不需要整理磁盘
  • 更易于了解计算机原理（如果你愿意）

相对而言，Linux 还有这些明显的不同：

  • 严格区分的用户和组，以及权限
  • 文件系统不同，不是以盘符分区
  • 文件区分大小写，并且包含权限
  • 不区分扩展名，而是以文件头内容识别
